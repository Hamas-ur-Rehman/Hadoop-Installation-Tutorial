Hadoop 3.4.1 Installation Guide (Ubuntu)

0%

Step 1: Install Java JDK 8

Hadoop requires Java to run.

            
            sudo apt install openjdk-8-jdk

Step 2: Set Up Environment Variables

            
            nano ~/.bashrc

Add these lines at the end of the file:

            
            export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
            export HADOOP_HOME=~/hadoop-3.4.1/
            export PATH=$PATH:$HADOOP_HOME/bin:$HADOOP_HOME/sbin

Then, apply the changes:

            
            source ~/.bashrc

Step 3: Download and Extract Hadoop

Download Hadoop 3.4.1 from the official site.

            
            tar -zxvf ~/Downloads/hadoop-3.4.1.tar.gz

Step 4: Configure Hadoop

Edit `hadoop-env.sh`:

            
            sudo nano hadoop-3.4.1/etc/hadoop/hadoop-env.sh

Set JAVA_HOME:

            
            JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64

Step 5: Set Up SSH

            
            ssh-keygen -t rsa -P '' -f ~/.ssh/id_rsa
        
            
            cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys

Step 6: Start Hadoop

            
            start-all.sh

Step 7: Run a WordCount Example

            
            hadoop jar $HADOOP_HOME/share/hadoop/mapreduce/hadoop-mapreduce-examples-3.4.1.jar wordcount /input /output

View the results:

            
            hadoop fs -cat /output/part-r-00000
